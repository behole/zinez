## üí° Lessons Learned

### What Worked:
1. **Collection-specific searches** - Most accurate results
2. **Exact zine names** - "Maximum Rocknroll" = 100% accuracy
3. **Established archives** - punkplanet, maximumrnr collections are gold
4. **Batch processing** - Efficient for large-scale expansion

### What Needs Work:
1. **Generic terms** - "hardcore" is too broad
2. **Boolean operators** - Need more sophisticated filtering
3. **Mediatype filtering** - Essential to exclude audiobooks
4. **Post-processing** - Quality review and cleanup needed

### Improvements for Next Run:
1. Add `mediatype:texts` or `collection:zines` to all queries
2. Exclude `librivox`, `audiobook`, `software` explicitly
3. Focus on known good collections first
4. Implement automated false positive detection
