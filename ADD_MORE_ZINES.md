## ID Naming Convention

Use these prefixes for IDs:
- **SG** = Sniffin' Glue
- **MRR** = Maximum Rocknroll
- **PP** = Punk Planet
- **BK** = Bikini Kill
- **FS** = Flipside
- **RT** = Ripped & Torn
- **PE** = Profane Existence
- **TD** = Touch and Go
- **FE** = Forced Exposure
- **HC** = Homocore
- **CB** = Cometbus
- **YF** = Your Flesh

Then add issue number: MRR001, MRR002, etc.

## Image Sources

### Where to Find Cover Scans:

1. **Direct from Archives**
   - Internet Archive (download page images)
   - Flickr collections
   - Museum digital collections

2. **Collector Sites**
   - eBay listings (for reference)
   - Discogs (record label zines)
   - Pinterest punk collections

3. **Academic Sources**
   - JSTOR (academic articles often include covers)
   - University digital collections
   - Thesis appendices

### Image Processing Tips:

1. **Ideal specs:**
   - 800px width minimum
   - JPG format
   - Keep original aspect ratio
   - Optimize for web (under 500KB)

2. **Naming:**
   - Use lowercase
   - Replace spaces with underscores
   - Include issue number

3. **Fallback:**
   - Create text-only covers for missing images
   - Use period-appropriate fonts
   - Match original color schemes
